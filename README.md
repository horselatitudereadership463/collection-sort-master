## 📑 Frequently Asked Questions

Does this tool delete my files?
No. This tool only moves files into new folders. It does not perform destructive actions on your data.

Can I sort files in multiple folders at once?
Currently, the tool supports single folder selection. You must repeat the process for each directory you wish to organize.

Does this work on network drives?
Yes. You can select mapped network drives as your target path. Ensure your connection remains stable during the entire sorting operation to prevent file errors.

Does the software support file renaming?
The current version focuses strictly on moving files into organized folders. It does not change file names, which ensures your file metadata remains exactly as it was when the file arrived on your system.

What happens if I try to sort a folder that is already organized?
The application detects the existing structure. It will skip files that are already in the correct location and will not create duplicate folders. 

Can I change where the tool places the sorted folders?
The tool creates sorted folders within the directory you select. It maintains the location of your original source files to ensure you can find them easily later.
